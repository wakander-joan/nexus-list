# 🚀 Nexus-List

**Nexus-List** é uma aplicação centralizada projetada para unir a gestão de **compras** e de **tarefas** em um único ponto de conexão. O elo perfeito entre o que você precisa ter e o que você precisa fazer.

---

### ✨ Funcionalidades Principais
* 🛒 **Gestão de Compras:** Organize itens, quantidades e categorias para suas idas ao mercado.
* ✅ **Gestão de Tarefas:** Liste seus afazeres diários com sistema de check e prioridades.
* 🎯 **Interface Unificada:** Navegação rápida e intuitiva entre listas de consumo e produtividade.

### 🛠️ Tecnologias Utilizadas
* **Linguagem:** Java 21
* **Framework:** Spring Boot 4.0.3
* **Persistência:** Spring Data JPA
* **Segurança:** Spring Security
* **Auxiliares:** Lombok & Bean Validation
